# Clone Raiz Airbnb - Mapa de hospedagens
Clone da página de busca por hospedagens do Airbnb, onde exibe um mapa e possui as seguintes funcionalidades:
- Mapa com preços e localizações das hospedagens
- Filtra cards de hospedagens de acordo com posicionamento do mapa
- Galeria de fotos nos cards das hospedagens

O código desse projeto foi criado nas lives raiz do canal Programador a Bordo.

Link das aulas (podem estar como privado):
- [Clone Raiz: Mapa de hotéis do Airbnb - parte 1 [Live #42]](https://youtu.be/9yPhP-SermA)

## Como rodar
Utilize versão Node mais recente

Comandos
```
npm install
```
```
npx serve
```
